# [PRD] 하이엔드 감성 조명 브랜드 '레비테 (Levité)' 홍보 랜딩 페이지 구축 명세서 (v1.4_Final)

본 문서는 28~39세 전문직 직장인 및 크리에이터를 타깃으로 한 하이엔드 키네틱 조명 브랜드 **'레비테 (Levité)'**의 스토리텔링형 홍보 랜딩 페이지의 기술 및 기획 요구사항을 정의합니다. 본 페이지는 스크롤 진행도와 유기적으로 결합된 하드웨어 가속 기반의 '이클립스 명도 반전'과 '키네틱 폴딩 및 인터랙티브 3D 셰이더 모션'을 통해 방문자에게 압도적인 해방감과 미학적 경외감을 선사하는 것을 목적으로 합니다.

---

## 1. 프로젝트 개요

- **프로젝트명:** 레비테 브랜드 스토리텔링 랜딩 페이지 개발
- **주요 타깃:** 28~39세 남녀 공통 (페르소나: 고도의 디지털 피로감을 겪으며 아날로그적 휴식과 미니멀 테크 미학을 추구하는 전문직 크리에이터 한진우)
- **핵심 슬로건 (Hero):** "빛이 형태를 바꾸어 흐를 때, 당신의 지친 공간은 영감이 됩니다."
- **최종 전환 CTA:** `[ 나만의 고요 구하기 ]`
- **방문자 타깃 감정:** 압도적인 해방감과 경외감 (Overwhelming Liberation & Awe)
- **기술 스택:** React + Vite, Vanilla CSS (컴포넌트 단위 스타일링), Intersection Observer API (뷰포트 감지), React Scroll State Binding (CSS 커스텀 변수 제어)

## 2. 디자인 시스템 및 디자인 자산 (Design System)

### 2.1 색상 팔레트 (Color Palette - 가변형 시스템)

- **배경색 (Background):**
  - `Warm Sand Ivory` (`#F7F4F0`) - 낮/라이트 모드 구역용. 모니터의 강한 빛을 흡수하는 눈이 편안한 아이보리 톤.
  - `Midnight Abyss` (`#0D1310`) - 밤/다크 모드 구역용. 빛의 윤슬 투사 효과를 극대화하기 위한 완벽한 암전 블랙 톤.
- **주조색 (Primary Text):**
  - `Deep Forest Chrome` (`#1B2B22`) - 라이트 배경용 텍스트 컬러. 차분하고 밀도 높은 딥 그린 가독성 확보.
  - `Pale Sage` (`#D1DCD6`) - 다크 배경용 반전 텍스트 컬러. 화이트 색상을 배제하여 야간 시각적 피로도 최소화.
- **강조색 및 광원 (Accent & Light):**
  - `Hypnotic Amber Glow` (`#E29543`) - 필라멘트와 노을빛을 담은 카멜 앰버 컬러. 윤슬 그래픽 및 핵심 CTA 버튼에 적용.

### 2.2 타이포그래피 (Typography)

- **본문 및 이성 스펙용 (Body):** `프리텐다드 (Pretendard)` - 정밀 공학 사양 및 가독성을 위한 현대적인 고딕체.
- **강조 및 서사 카피용 (Title):** `프리텐다드 (Pretendard)` - 기계적인 메커니즘 룩과 미니멀 테크 감성을 극대화하기 위해 명조체 대신 본문과 동일한 폰트로 통일하여 일관된 룩 앤 필(Look & Feel)을 확보함.
  - _보정 가이드:_ 다크 모드 구역 진입 시 폰트의 획이 타들어 가며 얇아지는 텍스트 렌더링 현상을 방지하기 위해 CSS `text-shadow: 0 0 1px var(--color-pale-sage);` 속성을 필수 적용함.

## 3. 섹션별 상세 구성 (순서, 목적, 레이아웃, 요소)

### Section 01. Hero (인트로)

- **목적:** 디지털 자극을 차단하고 브랜드가 제안하는 키네틱 미학의 서사로 유저를 깊게 몰입시킴.
- **레이아웃:** 비대칭 밸런스의 풀스크린(100vh) 레이아웃. 전체 화면에 마우스를 따라 반응하는 이미지 트레일을 깔고, 텍스트 콘텐츠는 화면 좌측에 정갈하게 고정하며 우측에는 시네마틱 비디오 프레임을 배치하여 모던한 구도를 연출.
- **들어갈 요소:** 상단 미니멀 네비게이션 바(로고, 최소 메뉴), 메인 슬로건, 시네마틱 배경 비디오 및 스크롤 연동 이미지 트레일, 스크롤 유도 인디케이터.
- **헤드라인 카피:** "빛이 형태를 바꾸어 흐를 때, 당신의 지친 공간은 영감이 됩니다."

### Section 02. Problem & Sympathy (일상의 자극)

- **목적:** 끝없는 모니터 스크린과 마감 시간(타임라인)에 쫓기는 타깃 전문직들의 일상적 피로에 깊이 공감하고 대조군 형성.
- **레이아웃:** 넓은 아이보리 여백 중심의 1:1 대칭형 2단 그리드 레이아웃. 한쪽에는 텍스트 블록, 반대편에는 차가운 모니터 픽셀 소스들이 추상적으로 나열되는 형태.
- **들어갈 요소:** 스크롤 시 숫자가 동적으로 카운트업(CountUp)되며 커서에 반응하는 추상 인포그래픽 에셋, 번짐(Blur) 효과 및 암호 해독(Decrypted) 애니메이션으로 선명해지는 공감 서사 텍스트.
- **헤드라인 카피:** "오늘 하루, 당신의 눈이 견뎌낸 타임라인은 몇 번의 깜빡임이었나요?"

### Section 03. Product Value 1 (타임라인 쉐이드 - 이클립스 반전)

- **목적:** 핵심 차별점 노출. 스크롤 동작과 연동하여 세상과 차단되는 완벽한 암전 속에서 아날로그 조명이 점등되는 의식(Ritual)을 시각적으로 증명.
- **레이아웃:** 좌측 화면 고정(Sticky) 텍스트 구역, 우측 스크롤 연동형 WebGL 3D 조명 오브제 구역. 스크롤이 진행됨에 따라 화면 전체의 색상이 완전히 반전되는 구조.
- **들어갈 요소:** 마우스 드래그와 물리 엔진에 반응하는 인터랙티브 WebGL 3D 조명(`KineticLamp`), 명도 반전(Eclipse) 레이어, 가치 설명 서사 카피.
- **헤드라인 카피:** "빛을 물리적으로 닫는 순간, 비로소 당신만의 안전한 시간이 시작됩니다."

### Section 04. Product Value 2 (시네마틱 공간 투사 - 압도적 해방감)

- **목적:** 두 번째 스펙터클 연출. 방이라는 물리적 갇힘을 허물고 3단계로 전환되는 시네마틱 WebGL 연출을 통해 화면을 가득 채워 압도적인 해방감 선사.
- **레이아웃:** 전면 다크 모드 풀 레이아웃. 텍스트는 화면 정중앙에 배치되며, 전체 배경은 마우스와 스크롤에 반응하는 입체적인 WebGL 파동으로 채워지는 레이어링 구조.
- **들어갈 요소:** 스크롤 단계(Step 1~3)에 따라 순차 전환되는 3가지 인터랙티브 WebGL 셰이더 배경(`Galaxy`, `Orb`, `LiquidChrome`), 페이드인(Fade-in) 되는 중앙 서사 텍스트 및 스펙 칩.
- **헤드라인 카피:** "어둠이 가라앉은 깊은 침묵 속에서, 일렁이는 조형 예술이 깨어납니다."

### Section 05. Trust & Engineering (디테일 & 빌드 퀄리티)

- **목적:** 테크 미니멀리스트인 타깃에게 부품의 정밀함, 무소음 설계, 압도적인 마감 퀄리티를 이성적(프리텐다드 폰트)으로 증명하여 신뢰 부여.
- **레이아웃:** 정돈된 다단 그리드 정보 레이아웃. 조명의 내부 구조 단면도(Exploded View)와 기술적 스펙 칩(Chip)들이 테크니컬하게 배열됨.
- **들어갈 요소:** 렌더링 이미지 대신 가벼운 벡터(SVG)로 직조된 조명 내부 단면도, 커서에 반응하는 정밀 스펙 팩트 시트 및 마감 소재 칩(Chip).
- **헤드라인 카피:** "숨겨진 태엽의 움직임까지 완벽하도록, 오직 아날로그 정밀함으로 조각했습니다."

### Section 06. Close & CTA (최종 전환)

- **목적:** 시각적 해방감의 여운을 구매 액션으로 유도하는 최종 전환 영역.
- **레이아웃:** 센터 정렬의 극도로 미니멀한 레이아웃. 앰버 빛 광원이 패키지를 투과하는 선(Outline) 형태의 SVG 애니메이션 일러스트를 배경으로 최종 CTA 버튼 배치.
- **들어갈 요소:** 프리미엄 언박싱 패키지 비주얼(SVG), 최종 클로징 카피, 호버 시 강조색(`#E29543`)이 채워지는 고스트(Ghost) 형태의 CTA 전환 버튼.
- **헤드라인 카피:** "오늘 밤, 끝없는 자극의 세계에서 완벽하게 퇴근하세요."
- **CTA 버튼 텍스트:** `[ 나만의 고요 구하기 ]`

## 4. 섹션별 모션 플로우 및 인터랙션 포인트

### 4.1 모션 플로우 (Motion Flow)

- **Hero Section:**
  - _진입 시:_ 배경 아이보리가 채워진 후 텍스트와 우측 시네마틱 비디오 프레임이 부드럽게 정위치에 안착하며 페이드인됩니다.
  - _마우스/스크롤 시:_ 마우스의 이동 경로를 따라 반응하며 화면 전역에 잔상(Image Trail)을 남기는 인터랙션이 구동됩니다.
- **Problem & Sympathy Section:**
  - _진입/스크롤 시:_ 서사 텍스트가 번짐(Smudge) 효과로 등장하며, 우측 노이즈 인포그래픽이 순차 페이드인 됨과 동시에 목표 수치까지 동적으로 카운트업(CountUp) 됩니다. 하단의 질문 텍스트는 암호가 해독(Decrypted)되는 연출로 나타납니다.
- **Product Value 1 (Timeline Shade) Section:**
  - _스크롤 시:_ 고유의 뷰포트 진입 진행도(`scrollRatio`)와 결합되어, 스크롤을 내리면 배경이 원형(Eclipse)으로 완벽히 암전됩니다. 이와 동시에 중앙의 WebGL 3D 조명(`KineticLamp`)에서 앰버 빛이 서서히 켜집니다.
  - _드래그/호버 시:_ 물리 엔진(Rapier)이 적용된 3D 램프 갓을 마우스로 직접 잡고 흔들 수 있는 피지컬 인터랙션이 발생합니다.
- **Product Value 2 (시네마틱 공간 투사) Section:**
  - _스크롤/터치 시:_ 3단계의 스크롤 위치(`stepIndex`)에 따라 우주 파티클(`Galaxy`), 빛의 구체(`Orb`), 유체 셰이더(`LiquidChrome`)라는 서로 다른 3개의 거대 WebGL 배경이 크로스페이드(Cross-fade)되며 전환됩니다.
- **Trust & Engineering Section:**
  - _진입/호버 시:_ 조명 단면도(SVG)와 스펙 칩들이 아래에서 위로 순차적(Fade-in-up)으로 등장하며, 칩 위로 마우스를 올리면 전용 커서(`⊙`)가 맞춤형으로 반응합니다.
- **Close & CTA Section:**
  - _진입 시:_ 빛이 퍼져나가는 선(Outline) 형태의 SVG 언박싱 패키지 애니메이션이 그려지며 클로징 카피가 선명해집니다.
  - _호버 시 (CTA 버튼):_ 고스트(Ghost) 버튼에 마우스를 올리면 앰버 빛이 부드럽게 내부를 채우는(ScaleX) 하이엔드 글로우 모션이 적용됩니다.

### 4.2 인터랙션 포인트 목록 (Interaction Points)

1. **브라우저 스크롤 스무딩 (Scroll Smoothing):** 전체 스크롤 휠 조작 시 렌더링이 끊기지 않고 60fps로 감속 미끄러지듯 이동하는 하드웨어 가속 휠 인터랙션 필수 적용.
2. **맞춤형 반응 커서 (Custom Cursor):** `mix-blend-mode: difference`가 적용된 기본 도트 커서가, 인터랙티브 요소에 닿을 때마다 브랜드 모티프인 정밀한 기하학적 형태(`⊙` 모양: 42px 원형 테두리 및 중앙 6px 코어)로 트랜지션되며 시각적 피드백 제공.
3. **물리 엔진 기반 실시간 렌더링 (Physics-based Interaction):** Section 03의 3D 조명 오브제(`KineticLamp`)에 물리 엔진(Rapier)을 결합하여, 마우스 드래그 시 실제 질량과 관성을 가진 물리적 흔들림 및 스프링 바운스 모션을 실시간으로 완벽하게 구현.

## 5. 모바일 대응 방향 (Responsive Design)

- **인터랙션 경량화 (Performance):** 모바일(768px 미만) 환경의 렌더링 과부하를 방지하기 위해 맞춤형 반응 커서 레이어(`Custom Cursor`)는 모바일 진입 시 `display: none`으로 전면 비활성화하여 쾌적한 스크롤 성능을 확보합니다.
- **레이아웃 가변 그리드:** 데스크톱의 2단 스티키 그리드나 복합 스펙표(Trust & Engineering)는 모바일 화면에서 단일 열(Single Column) 상하 구조로 변환됩니다. 거대한 WebGL 조명 오브제 영역 등은 서사 텍스트가 오브제 하단에 최적화되어 배치되는 뷰를 제공합니다.
- **타이포그래피 및 WebGL 최적화:** 전면 적용된 'Pretendard' 단일 폰트의 모바일 가독성을 위해 최소 폰트 크기 가이드라인(`1.1rem`)을 방어하며, 모바일 기기에서의 과도한 배터리 소모 및 발열을 방지하기 위해 물리 연산(Rapier) 파라미터나 WebGL 셰이더 품질을 유동적으로 최적화하는 방향으로 대응합니다.

## 6. 기술 구현 가이드 (React + Vite + Vanilla CSS)

### 6.1 아키텍처 및 디자인 인프라 변수 세팅 (variables.css)

```css
/* src/styles/variables.css */
:root {
  /* 라이트 모드 자산 */
  --color-bg-ivory: #F7F4F0;
  --color-text-forest: #1B2B22;

  /* 다크 모드 자산 (이클립스 반전용) */
  --color-bg-abyss: #0D1310;
  --color-text-sage: #D1DCD6;

  /* 핵심 광원 및 강조색 */
  --color-accent-amber: #E29543;

  /* 폰트 시스템 */
  --font-body: 'Pretendard', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
  --font-title: 'Pretendard', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
}
```
