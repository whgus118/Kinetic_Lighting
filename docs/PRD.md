# [PRD] 하이엔드 감성 조명 브랜드 '레비테 (Levité)' 홍보 랜딩 페이지 구축 명세서 (v1.4_Final)

본 문서는 28~39세 전문직 직장인 및 크리에이터를 타깃으로 한 하이엔드 키네틱 조명 브랜드 **'레비테 (Levité)'**의 스토리텔링형 홍보 랜딩 페이지의 기술 및 기획 요구사항을 정의합니다. 본 페이지는 스크롤 진행도와 유기적으로 결합된 하드웨어 가속 기반의 '이클립스 명도 반전'과 '3D 디컨스트럭션 모션'을 통해 방문자에게 압도적인 해방감과 미학적 경외감을 선사하는 것을 목적으로 합니다.

---

## 1. 프로젝트 개요

- **프로젝트명:** 레비테 브랜드 스토리텔링 랜딩 페이지 개발
- **주요 타깃:** 28~39세 남녀 공통 (페르소나: 고도의 디지털 피로감을 겪으며 아날로그적 휴식과 미니멀 테크 미학을 추구하는 전문직 크리에이터 한진우)
- **핵심 슬로건 (Hero):** "빛이 형태를 바꾸어 흐를 때, 당신의 지친 공간은 영감이 됩니다."
- **최종 전환 CTA:** `[ 나만의 고요 구하기 ]`
- **방문자 타깃 감정:** 압도적인 해방감과 경외감 (Overwhelming Liberation & Awe)
- **기술 스택:** React + Vite, Vanilla CSS (컴포넌트 단위 스타일링), Intersection Observer API (뷰포트 감지), React Scroll State Binding (CSS 커스텀 변수 제어)

## 2. 디자인 시스템 및 디자인 자산 (Design System)

### 2.1 색상 팔레트 (Color Palette - 가변형 시스템)

- **배경색 (Background):**
  - `Warm Sand Ivory` (`#F7F4F0`) - 낮/라이트 모드 구역용. 모니터의 강한 빛을 흡수하는 눈이 편안한 아이보리 톤.
  - `Midnight Abyss` (`#0D1310`) - 밤/다크 모드 구역용. 빛의 윤슬 투사 효과를 극대화하기 위한 완벽한 암전 블랙 톤.
- **주조색 (Primary Text):**
  - `Deep Forest Chrome` (`#1B2B22`) - 라이트 배경용 텍스트 컬러. 차분하고 밀도 높은 딥 그린 가독성 확보.
  - `Pale Sage` (`#D1DCD6`) - 다크 배경용 반전 텍스트 컬러. 화이트 색상을 배제하여 야간 시각적 피로도 최소화.
- **강조색 및 광원 (Accent & Light):**
  - `Hypnotic Amber Glow` (`#E29543`) - 필라멘트와 노을빛을 담은 카멜 앰버 컬러. 윤슬 그래픽 및 핵심 CTA 버튼에 적용.

### 2.2 타이포그래피 (Typography)

- **본문 및 이성 스펙용 (Body):** `프리텐다드 (Pretendard)` - 정밀 공학 사양 및 가독성을 위한 현대적인 고딕체.
- **강조 및 서사 카피용 (Title):** `이롭게 바탕 (Iropke Batang)` - 획의 끝이 직선적이고 샤프하여 조명의 기계적 메커니즘 실루엣과 완벽한 싱크를 이루는 웹 스크린 최적화 명조체.
  - _보정 가이드:_ 다크 모드 구역 진입 시 명조체의 획이 타들어 가며 얇아지는 현상을 방지하기 위해 CSS `text-shadow: 0 0 1px var(--color-pale-sage);` 속성을 필수 적용함.

## 3. 섹션별 상세 구성 (순서, 목적, 레이아웃, 요소)

### Section 01. Hero (인트로)

- **목적:** 디지털 자극을 차단하고 브랜드가 제안하는 키네틱 미학의 서사로 유저를 깊게 몰입시킴.
- **레이아웃:** 풀스크린(100vh) 미니멀리즘 레이아웃. 중앙에 여백을 크게 두고 조명 제품의 미니멀한 실루엣 이미지를 배치하며, 텍스트는 정갈하게 정렬.
- **들어갈 요소:** 상단 미니멀 네비게이션 바(로고, 최소 메뉴), 메인 슬로건, 제품 실루엣 그래픽, 스크롤 유도 인디케이터.
- **헤드라인 카피:** "빛이 형태를 바꾸어 흐를 때, 당신의 지친 공간은 영감이 됩니다."

### Section 02. Problem & Sympathy (일상의 자극)

- **목적:** 끝없는 모니터 스크린과 마감 시간(타임라인)에 쫓기는 타깃 전문직들의 일상적 피로에 깊이 공감하고 대조군 형성.
- **레이아웃:** 넓은 아이보리 여백 중심의 비대칭 그리드 레이아웃. 한쪽에는 텍스트 블록, 반대편에는 차가운 모니터 픽셀 소스들이 추상적으로 나열되는 형태.
- **들어갈 요소:** 일상의 디지털 소음을 비주얼화한 추상 인포그래픽 에셋, 공감 서사 본문 텍스트.
- **헤드라인 카피:** "오늘 하루, 당신의 눈이 견뎌낸 타임라인은 몇 번의 깜빡임이었나요?"

### Section 03. Product Value 1 (타임라인 쉐이드 - 이클립스 반전)

- **목적:** 핵심 차별점 노출. 스크롤 동작과 연동하여 물리적으로 조명 갓이 닫히며 세상과 차단되는 아날로그적 의식(Ritual)을 시각적으로 증명.
- **레이아웃:** 좌측 화면 고정(Sticky) 텍스트 구역, 우측 스크롤 연동형 CSS 3D 조명 오브제 구역. 스크롤이 진행됨에 따라 화면 전체의 색상이 완전히 반전되는 구조.
- **들어갈 요소:** 3D 구조의 조명 갓 가동 이미지 시퀀스, 명도 반전 레이어, 가치 설명 서사 카피.
- **헤드라인 카피:** "빛을 물리적으로 닫는 순간, 비로소 당신만의 안전한 시간이 시작됩니다."

### Section 04. Product Value 2 (빛의 윤슬 투사 - 경외감)

- **목적:** 두 번째 스펙터클 연출. 방이라는 물리적 갇힘을 허물고 대자연의 황홀경(윤슬)으로 천장과 화면을 가득 채워 압도적인 해방감 선사.
- **레이아웃:** 전면 다크 모드(`Midnight Abyss`) 풀 레이아웃. 텍스트가 화면 밑바닥에 잔잔히 깔리고 화면 전체가 빛의 파동으로 물드는 시네마틱 레이어링.
- **들어갈 요소:** 앰버 빛(`#E29543`)의 움직이는 윤슬 레이어(CSS Gradient 중첩), 은은한 빛의 투사 시각 자산.
- **헤드라인 카피:** "단조롭던 천장에 일렁이는 바다를 띄워, 방의 경계를 영원히 허뭅니다."

### Section 05. Trust & Engineering (디테일 & 빌드 퀄리티)

- **목적:** 테크 미니멀리스트인 타깃에게 부품의 정밀함, 무소음 설계, 압도적인 마감 퀄리티를 이성적(프리텐다드 폰트)으로 증명하여 신뢰 부여.
- **레이아웃:** 정돈된 다단 그리드 정보 레이아웃. 조명의 내부 구조 단면도(Exploded View)와 기술적 스펙 칩(Chip)들이 테크니컬하게 배열됨.
- **들어갈 요소:** 조명 메커니즘 정밀 렌더링 컷, 노이즈 없는 무소음 모터 사양 정보, 천연석/알루미늄 가공 마감 스펙 팩트 시트.
- **헤드라인 카피:** "숨겨진 태엽의 움직임까지 완벽하도록, 오직 아날로그 정밀함으로 조각했습니다."

### Section 06. Close & CTA (최종 전환)

- **목적:** 시각적 해방감의 여운을 구매 액션으로 유도하는 최종 전환 영역.
- **레이아웃:** 센터 정렬의 극도로 미니멀한 레이아웃. 앰버 빛 광원이 패키지를 투과하는 고급스러운 브랜드 컷을 배경으로 최종 CTA 버튼 배치.
- **들어갈 요소:** 프리미엄 언박싱 패키지 비주얼, 최종 클로징 카피, 강조색(`#E29543`)이 전면 적용된 최종 CTA 전환 버튼.
- **헤드라인 카피:** "오늘 밤, 끝없는 자극의 세계에서 완벽하게 퇴근하세요."
- **CTA 버튼 텍스트:** `[ 나만의 고요 구하기 ]`

## 4. 섹션별 모션 플로우 및 인터랙션 포인트

### 4.1 모션 플로우 (Motion Flow)

- **Hero Section:**
  - _진입 시:_ 배경 아이보리가 채워진 후 메인 카피가 블러 처리(`filter: blur(12px) -> 0px`)에서 부드럽게 깨어나며 1.8초간 페이드인됩니다. 중앙 조명 실루엣이 미세하게 위로 올라옵니다.
  - _스크롤 시:_ 사용자가 스크롤을 시작하면 글자 간격(letter-spacing)이 미세하게 넓어지며 위로 페이드아웃됩니다.
  - _호버 시:_ 스크롤 다운 인디케이터에 마우스를 올리면 아래로 물방울이 떨어지듯 잔잔한 바운스 키프레임 애니메이션 속도가 가속됩니다.
- **Problem & Sympathy Section:**
  - _진입/스크롤 시:_ 스크롤 트리거에 의해 문장이 줄 단위로 타임 딜레이(0.2s씩)를 두고 안개 속에서 글자가 번져 나오듯(Fade & Smudge) 뷰포트에 등장합니다.
- **Product Value 1 (Timeline Shade) Section: [핵심 모션 구역]**
  - _스크롤 시:_ 전체 페이지 기준이 아닌 **해당 섹션 고유의 뷰포트 진입 진행도**(`--section-scroll-ratio`)와 1:1로 결합되어 구동됩니다. 스크롤을 내릴 때 조명 갓 이미지 레이어가 기하학적으로 접혀 닫힙니다. 이와 동시에 CSS `clip-path` 또는 투명도 제어를 통해 배경색이 `#F7F4F0`에서 완전한 어둠인 `#0D1310`으로 완전히 일식(Eclipse)되듯 슬라이딩 전환됩니다.
- **Product Value 2 (윤슬 투사) Section:**
  - _진입/스크롤 시:_ 암전이 완료된 후, 배경 뒤에 대기하던 앰버 빛 그라디언트 맵들이 `mix-blend-mode: screen` 상태로 서서히 살아나며 찰랑거립니다. 텍스트는 아래서 위로 번지듯 등장합니다.
- **Trust & Engineering Section:**
  - _진입 시:_ 복잡한 감성 모션을 배제하고, 정밀한 부품 레이아웃들이 왼쪽에서 오른쪽으로 정갈하고 절제되게 슬라이드인(Slide-in)되어 정보의 가독성을 높입니다.
- **Close & CTA Section:**
  - _진입 시:_ 최종 패키지 비주얼 위로 클로징 카피가 선명해지며, CTA 버튼이 은은하게 숨쉬듯(Pulse) 깜빡입니다.
  - _호버 시 (CTA 버튼):_ `[ 나만의 고요 구하기 ]` 버튼에 마우스를 올리면 버튼 내부의 카멜 앰버 빛이 바깥 영역으로 부드럽게 번져나가는 하이엔드 글로우(Glow) 효과가 마우스 아웃 시까지 지속됩니다.

### 4.2 인터랙션 포인트 목록 (Interaction Points)

1. **브라우저 스크롤 스무딩 (Scroll Smoothing):** 전체 스크롤 휠 조작 시 렌더링이 끊기지 않고 60fps로 감속 미끄러지듯 이동하는 하드웨어 가속 휠 인터랙션 필수 적용.
2. **커서 추적 앰버 글로우 (Cursor Follower):** 다크 모드 구역(Section 03, 04)에서 사용자의 마우스 커서를 반 박자 늦게 따라다니는 지연(Delay) 방식의 은은한 원형 앰버 조명 필터 효과 레이어를 결합하여 마우스 무브먼트에 화려한 반응성 제공.
3. **실시간 스크롤 동기화 매커니즘 (Scrollbound Target):** Section 03의 조명 갓의 폴딩 회전 각도 값(`transform: rotateX()`)을 마우스 스크롤 트랙킹과 완벽 동기화.

## 5. 모바일 대응 방향 (Responsive Design)

- **스크롤 애니메이션 최적화 및 경량화:** 모바일(768px 미만) 환경의 GPU 오버헤드를 예방하기 위해 연산량이 많은 `filter: blur()` 값의 반경을 50% 축소하고, 마우스 커서 추적 레이어는 모바일 진입 시 `display: none`으로 즉시 전면 비활성화하여 스크롤 버벅임을 차단합니다.
- **레이아웃 가변 그리드:** 데스크톱의 2열 스티키 그리드(좌측 서사 고정, 우측 3D 변형 구조)는 모바일 화면에서 **단일 열(Single Column) 상하 스크롤 구조**로 자동 변환됩니다. 조명 오브제가 상단에 고정 배치되고, 그 아래로 서사 카피들이 오버레이되며 올라오는 형식으로 대체합니다.
- **타이포그래피 및 인터랙션 제한:** 명조체(이롭게 바탕) 특성상 모바일 화면에서 크기가 작아지면 가독성이 파괴되므로, 최소 폰트 크기 가이드라인(`1.1rem`)을 지정하고 데스크톱 대비 `letter-spacing`을 1% 넓혀 가독성을 방어합니다. 또한 미디어 쿼리를 사용해 모바일 브레이크포인트에서는 과도한 3D 회전 연산이 일어나지 않도록 예외 처리합니다.

## 6. 기술 구현 가이드 (React + Vite + Vanilla CSS)

### 6.1 아키텍처 및 디자인 인프라 변수 세팅 (variables.css)

```css
/* src/styles/variables.css */
:root {
  /* 라이트 모드 자산 */
  --color-bg-ivory: #f7f4f0;
  --color-text-forest: #1b2b22;

  /* 다크 모드 자산 (이클립스 반전용) */
  --color-bg-abyss: #0d1310;
  --color-text-sage: #d1dcd6;

  /* 핵심 광원 및 강조색 */
  --color-accent-amber: #e29543;

  /* 폰트 시스템 */
  --font-body: "Pretendard", -apple-system, sans-serif;
  --font-title: "Iropke Batang", serif;
}

body {
  font-family: var(--font-body);
  background-color: var(--color-bg-ivory);
  color: var(--color-text-forest);
  -webkit-font-smoothing: antialiased;
  overflow-x: hidden;
  margin: 0;
  padding: 0;
}
```
